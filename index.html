⚙️Config & Server Switching4

Change server, name, or secret key without rebuilding the exe — just send a command.

update_serverKEYChange server URL without rebuilding — works even mid-switch▶
Usage
update_server wss://new-app.up.railway.app/ws
Switching Railway accounts
/send script1 update_server wss://my-new-server.up.railway.app/ws
How it works: Saves the new URL to config.json immediately. The script reconnects to the new server automatically. Send this command on the OLD server before you shut it down — scripts will follow you to the new one.
update_secretNEWChange the secret key without rebuilding▶
Usage
update_secret NewSecretKey123
⚠️ Update the server env variable at the same time, or the script won't reconnect.
update_nameNEWRename this PC without rebuilding▶
Usage
update_name brothers_gaming_pc
Saved to config.json. Restart the app to reconnect with the new name.
show_configNEWShow current server URL, PC name, secret key▶
Usage

⌨️Keyboard Control3
type_textFIXEDType in ANY language — Russian, Arabic, Chinese, etc.▶
Usage
type_text text in any language
All languages work
/send script1 type_text Hello World ← English
/send script1 type_text Привет мир ← Russian ✓
/send script1 type_text مرحبا بالعالم ← Arabic ✓
/send script1 type_text 你好世界 ← Chinese ✓
Why it now works: pyautogui.write() only handles ASCII. The fix puts text in the clipboard and pastes with Ctrl+V — works for every language.
⚠️ Click the text field first with mouse_click, then use type_text.
